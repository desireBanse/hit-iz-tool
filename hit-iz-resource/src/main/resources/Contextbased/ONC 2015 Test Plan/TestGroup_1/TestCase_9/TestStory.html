Description
A patient is seen at a clinic. A new administration is recorded. The IIS is able to process the administration message but finds multiple soft errors (warnings) which it returns in the acknowledgement message.&nbsp;

Comments
No Comments

Pre Condition
No PreCondition

Post Condition
The returned error must be visible in the EHR.

Test Objectives
This test case assesses the ability of the EHR to accept an acknowledgement message with multiple soft errors being returned by the IIS.

Evaluation Criteria
No evaluation criteria
